# Claude 작업 가이드

## 커밋/푸시 시 자동 작업

매일 커밋/푸시를 할 때마다 다음 작업을 수행해주세요:

1. **박진석_개발일지.md 업데이트**
   - 오늘 날짜로 새 항목 추가
   - 커밋된 작업 내용을 간단히 기록
   - 형식: `## YYYY-MM-DD` 아래에 불렛 포인트로 작업 내용 나열

2. **커밋 후 개발일지도 함께 커밋**
   - 변경사항 커밋 → 개발일지 업데이트 → 개발일지 커밋 → 푸시

---

**예시:**
```markdown
## 2025-12-16

- Unity 프로젝트 생성 (URP)
- Git 저장소 초기화 및 GitHub 연동
- README 작성

## 2025-12-17

- 미로 생성 시스템 구현 시작
- 플레이어 이동 컨트롤러 작성
```
